Course introduction

What is this course about?

A hands-on, project-based program to learn data analysis and reproducible research using R and the tidyverse.

You will learn to:

  • Explore, transform, and visualize data
  • Write reproducible reports with Quarto
  • Use version control with Git and GitHub
  • Apply open science and FAIR data principles
  • Build a data analysis portfolio

Git & GitHub

What is Git?

Git is a version control system that tracks changes in your files over time.

Think of it as an unlimited undo button for your projects – you can go back to any previous version, see what changed, and work on different ideas without losing anything.

What is GitHub?

GitHub is a web platform that hosts Git repositories online.

It allows you to:

  • Share your code with others
  • Collaborate on projects
  • Showcase your work as a portfolio
  • Access other people’s code and data

Git = the tool on your computer. GitHub = the online platform.

R

What is R?

R is a programming language designed for data analysis, statistics, and visualization.

Why R?

  • Free and open source
  • Thousands of packages for every type of analysis
  • Excellent for creating plots and reports
  • Large and supportive community
  • Widely used in academia and industry

We will use RStudio as our development environment – it makes working with R much easier.

Quarto

What is Quarto?

Quarto is an open-source publishing system that lets you combine code, text, and results into a single document.

You can create:

  • Reports and articles
  • Presentations (like these slides!)
  • Websites and blogs
  • Books

Why does this matter?

Quarto enables reproducible research: anyone can re-run your analysis and get the same results. No more copy-pasting figures into Word!
